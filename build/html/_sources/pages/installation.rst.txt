.. _installation:

============
Installation
============

* clone the project from github:

ssh:

.. code:: bash

    git clone git@github.com:universalcustomer/chatbot-universal.git

https:

.. code:: bash

    git clone https://github.com/universalcustomer/chatbot-universal.git

* you need to have installed `composer <https://getcomposer.org/>`_ and `node.js <https://docs.npmjs.com/downloading-and-installing-node-js-and-npm>`_

* install dependencies from composer:

.. code:: bash

    composer install

* install npm dependencies:

.. code:: bash

    npm install

* Install npm package laravel-echo-server globally with the following command:

.. code:: bash

    npm install -g laravel-echo-server

* install `Redis <https://redis.io/download>`_

* create chatbot in `Telegram <https://core.telegram.org/bots#6-botfather>`_

* create database( I use mysql)

* create a project(chatbot) inside `Dialogflow <https://dialogflow.cloud.google.com/>`_
